There is an ANT script for building this project, in the same directory as this readme file.

OS specifications :
	- Tested with Windows 7 and Mac OS X
	- Our implementation and the libraries are also compatible with Linux systems, but we didn't try

Hardware needed :
	- Microsoft Kinect and OpenNI drivers installed
	- For drivers installation : http://code.google.com/p/simple-openni/downloads/list

Main class for launching program :
	- ProjectApplication.java (package "main")
	
This application is designed for Java SE 1.7 but should also be compatible with previous versions
